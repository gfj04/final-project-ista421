# Auto MPG Analysis (ISTA 421 Final Project)

## Overview
This project analyzes the relationship between vehicle characteristics and fuel efficiency (MPG) using regression models.

## Key Results
- Strong negative relationship between weight and MPG
- R² ≈ 0.65 using weight only
- Model improves when adding horsepower and displacement

## Methods
- Data cleaning using pandas
- Visualization using matplotlib
- Linear regression using sklearn

## Conclusion
Vehicle weight is a major factor affecting fuel efficiency, but combining multiple features improves prediction accuracy.
